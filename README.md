# Paths-Patterns
A collection of patterns and paths confirmed to be safe by the developers of Natro Macro.

## How to add your pattern/path
If you have any additional files (default settings, images, videos, pattern instructions, etc.), please add them inside of the issue or pull request.
### Pattern
To add a pattern to this repository, create an issue with the pattern issue template. A pull request will be automatically created, and then merged once your code has been reviewed.
### Path
To add a path to this repository, create an issue with the path issue template. You can add the files through the issue by changing the file extension to .txt, or by pasting the code content.

You can also open a PR yourself. Fork the repository, make the changes in the apropiate format and create a PR on this repository.

## Updating patterns/paths
To update a pattern/path, open an issue with the update template, then your pattern/path will be updated by a staff member as soon as possible.
